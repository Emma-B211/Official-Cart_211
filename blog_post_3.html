How I Worked on My Second Using UE4

This project was created during my first semester at Dawson College through a class called "Game Making" through using Unreal Engine 4.11.

For the course we had to create a game during the whole semester after learning the basic when it comes to using Unreal Engine 4. I had created a puzzle-platformer game based on a employee return to their workplace only to find it abandon and you are tasked to find out what had happen while something is lurking around and you have to solve puzzles to find out what happened and to escape the facility.


In the future, I would like to revamp and improve the game and make it have even more potential when I first started working on it during the 15 weeks I had to work on it. I would like more dialogue, actual npcs and creatures to further make it a puzzle, platformer game mixed with a horror genre.
